# Camper Cafe Menu

A responsive cafe menu website built with HTML and CSS.

## Features
- Responsive design that works on mobile and desktop
- Clean, visually appealing cafe menu layout
- Coffee and dessert sections with pricing
- Interactive links with hover effects
- Centered menu with coffee beans background

## Technologies Used
- HTML5
- CSS3
- Responsive design principles

## Project Structure
cafe-menu/  
├── index.html # Main HTML file  
├── styles.css # CSS styles  
└── README.md # This file  


## How to View
Open `index.html` in any modern web browser
